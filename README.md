# Blade & Co

Одностраничный лендинг барбершопа, созданный для учебного задания по публикации проекта на GitHub Pages.

## Публикация на GitHub Pages

1. Создайте публичный репозиторий на GitHub.
2. Загрузите в него файлы проекта.
3. Откройте `Settings` -> `Pages`.
4. В `Build and deployment` выберите `Deploy from a branch`.
5. Укажите ветку `main` и папку `/root`, затем нажмите `Save`.

После публикации GitHub покажет ссылку вида:

```text
https://ВАШ-ЛОГИН.github.io/ИМЯ-РЕПОЗИТОРИЯ/
```
